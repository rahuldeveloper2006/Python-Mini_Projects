# Contact Manager (Python Mini Project)

This is a simple Contact Manager project built using Python and JSON.

## Features
- Add new contact
- View all contacts
- Search contact
- Delete contact
- Contacts are stored in a JSON file

## Technologies Used
- Python
- JSON
- File Handling

## How to Run

1. Download or clone the repository
2. Go to the Contact_Manager Project folder
3. Run the program

```bash
python contact_manager.py
